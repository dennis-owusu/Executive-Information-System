# Executive-Information-System

A comprehensive Executive Information System (EIS) for e-commerce, featuring:
- Customer-facing shopping experience (React + Vite, Tailwind, Framer Motion)
- Secure backend APIs (Express + Mongoose), product management with uploads
- Executive dashboard with sidebar navigation, KPIs, product analytics, and management tools

This repository contains two apps:
- `client/` React frontend
- `backend/` Node.js/Express backend

Ensure you create your own environment variables in `backend/.env` (not committed) before running.
